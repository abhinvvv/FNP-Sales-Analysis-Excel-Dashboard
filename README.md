# FNP Sales Analysis Dashboard (Excel)

Interactive sales dashboard built using Excel Pivot Tables, Data Model relationships and slicers to analyze business performance.

## Objectives
- Track revenue trends
- Identify top products and categories
- Analyze seasonal demand
- Study customer ordering patterns

## Tools Used
- Excel
- Power Pivot (Data Model)
- Pivot Tables
- Slicers & Timeline
- Data Cleaning

## Key Insights
- Highest revenue during Holi & Anniversary occasions
- February and August show peak seasonal demand
- Colors category generates maximum revenue
- Orders higher on weekends
